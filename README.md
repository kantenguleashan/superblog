# Super Blog

Super Blog is a simple yet powerful blogging platform that enables users to create, edit, and manage blog posts with ease
. It includes features like authentication, post creation, and editing, with media uploads and user profile management.

## Features

- **User Authentication**: Register and log in to your account securely.
- **Create and Edit Posts**: Write and edit your blog posts.
- **Media Upload**: Upload images and files for your posts.
- **Responsive Design**: Optimized for both desktop and mobile viewing.

## Tech Stack

- **Frontend**: React, React Quill (for rich text editor), React Router
- **Backend**: Node.js, Express.js, MongoDB
- **Authentication**: JSON Web Tokens (JWT)
- **File Upload**: Multer for handling file uploads

## Installation

### Prerequisites

Ensure you have the following installed:

- [Node.js](https://nodejs.org/)
- [MongoDB](https://www.mongodb.com/)
- [npm](https://www.npmjs.com/)

### Backend Setup

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/super-blog.git
    cd super-blog/backend
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Set up your `.env` file with the following environment variables:
    - `MONGODB_URI`: Your MongoDB connection string
    - `JWT_SECRET`: Secret key for JWT authentication

4. Start the backend server:
    ```bash
    npm start
    ```

   The backend should now be running at `http://localhost:4000`.

### Frontend Setup

1. Navigate to the frontend directory:
    ```bash
    cd frontend
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Start the frontend development server:
    ```bash
    npm start
    ```

   The frontend should now be running at `http://localhost:3000`.

## Usage

1. Open the app in your browser and create an account or log in if you already have one.
2. Once logged in, you can create, edit, and delete blog posts.
3. Posts support media uploads such as images and files.
